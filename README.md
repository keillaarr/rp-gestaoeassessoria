##  Funcionalidades

- **Design Responsivo** – Layout adaptável para desktops, tablets e smartphones.  
- **Cards Interativos** – Hover effects e sombra para destaque visual.  
- **Formulário de Contato** – Envio via **FormSubmit** (sem backend).  
- **Tipografia Personalizada** – RoxboroughCF para títulos e Garet para textos.  
- **Cores Harmonizadas** – Paleta de laranja, cinza e preto para melhor contraste.  
- **Seções Claras e Objetivas** – Facilita navegação e compreensão dos serviços.

## Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/keillaarr/rp-gestaoeassessoria.git
cd rp-gestaoeassessoria
```

2. Abra o arquivo index.html no navegador.

3. O formulário de contato já está configurado via FormSubmit. Para alterar o e-mail de destino, abra index.html e edite o atributo action do formulário com seu link do FormSubmit:
```html
<form action="https://formsubmit.co/seuemail@dominio.com" method="POST">
```

## Deploy no GitHub Pages
Vá até o repositório no GitHub.

Clique em Settings > Pages.

Em Source, selecione a branch main e a pasta / (root).

Salve e aguarde alguns minutos.

O site estará disponível em:

````arduino
https://SEU_USUARIO.github.io/rp-gestaoeassessoria/
````


## Design e UX
Paleta de cores: laranja (#ff7a00), cinza claro (#e8e8e3) e preto (#1c1c1c)

Tipografia: RoxboroughCF (títulos) e Garet (textos)

Cards com bordas superiores destacadas e sombra leve para efeito de relevo

Seções com padding consistente para melhor leitura

Layout visual moderno, limpo e intuitivo
